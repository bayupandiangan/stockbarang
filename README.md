📦 StockBase Inventory System

Sistem manajemen inventaris berbasis web yang dirancang untuk membantu pengelolaan stok barang secara efisien.
Dilengkapi dengan fitur CRUD, autentikasi pengguna, dan dashboard responsif untuk monitoring data secara real-time.

🚀 Features
🔐 Authentication System (Login & Logout)
📦 Stock Management (CRUD)
Tambah barang
Edit data barang
Hapus barang
Lihat daftar stok
📊 Dashboard Overview
Ringkasan data inventaris
🔎 Stock Tracking
Monitoring jumlah stok
📱 Responsive UI
Tampilan optimal di desktop & mobile
🛠️ Tech Stack
Backend: PHP
Database: MySQL
Frontend: HTML, CSS, JavaScript
UI Framework: Bootstrap
